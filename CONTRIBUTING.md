# Guia de Contribuição
Agradecemos o seu interesse em contribuir para o projeto FilmFrame.
Siga estas diretrizes para tornar o processo de contribuição produtivo.

## Formas de Contribuição

Você pode contribuir:

- Relatando bugs e problemas
- Melhorando a documentação
- Adicionando novos recursos
- Dando dicas e sugestões
- Realizando testes e validações

## Primeiros passos

1. Crie um fork deste repositório.
2. Faça as alterações necessárias.
3. Realize commits significativos
4. Abra um Pull Request neste repositório, explicando as mudanças feitas.

## Regras de Codificação

- Siga o estilo de código do projeto.
- Use espaços em vez de tabs.
- Mantenha linhas com até 80 caracteres para facilitar a leitura.

## Commits

- Use mensagens de commit claras e concisas.
- Certifique-se de que todas as alterações estão cobertas por testes.
- Execute os testes existentes antes de abrir um Pull Request.

## Licença

Ao contribuir, você concorda que suas contribuições serão licenciadas sob a mesma licença do projeto.

## Agradecimento

Agradecemos a todos os contribuidores por suas valiosas contribuições para este projeto!
